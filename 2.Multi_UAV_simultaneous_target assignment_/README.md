## Quick Paper Summary

**Algorithm:** TANet-TD3 (Target Assignment Network + Twin-Delayed Deep Deterministic Policy Gradient)

**Problem solved:** Multiple drones simultaneously deciding which target to reach AND how to fly there — in real time — in a 3D environment with moving obstacles and limited sensor vision.

**Key result:** 84.27% target completion rate vs. 73–80% for all baselines; 5/5 targets reached in deployment tests (best baseline: 2/5 in hard conditions).

**Published:** Frontiers in Neurorobotics, January 2024
