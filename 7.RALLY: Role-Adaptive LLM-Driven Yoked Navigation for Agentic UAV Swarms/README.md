## Quick Paper Facts

- **Authors:** Ziyao Wang, Rongpeng Li, Sizhao Li, Yuming Xiang, Haiping Wang, Zhifeng Zhao, Honggang Zhang
- **Published:** IEEE Open Journal of Vehicular Technology, Volume 6, 2025
- **DOI:** 10.1109/OJVT.2025.3610852
- **Key algorithm:** RALLY = Two-stage LLM consensus + RMIX dynamic role assignment
- **Three roles:** Commander / Coordinator / Executor
- **Validated in:** Multi-Agent Particle Environment (MPE) + Gazebo-ROS-PX4 SITL
- **Deployed model:** Fine-tuned Qwen2.5-1.5B (2.9 GB, runs on onboard UAV GPU)
