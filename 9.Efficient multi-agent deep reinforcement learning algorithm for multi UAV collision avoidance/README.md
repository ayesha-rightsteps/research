## Quick Paper Facts

- **Algorithm proposed:** IGAT-MARL (Improved Graph Attention Multi-Agent Reinforcement Learning)
- **Problem solved:** Multi-UAV collision avoidance in dense shared airspace
- **Key innovation:** Conflict-driven dynamic interaction graph + stacked double-attention IGAT network
- **Top result:** 17.56% higher reward, 10.52% fewer dangerous near-miss events, 43.93% fewer interaction edges vs. DGN benchmark
- **Environment:** BlueSky simulator + BADA aircraft performance data
- **Training:** 10,000 episodes, 3 to 10 UAVs, NVIDIA A40 GPU
- **Published:** Applied Soft Computing, Vol. 197 (2026), Elsevier
