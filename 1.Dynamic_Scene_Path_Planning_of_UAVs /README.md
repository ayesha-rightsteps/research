## Quick Summary (30 seconds)

This paper teaches a drone to navigate around **moving** threat zones using an improved deep reinforcement learning algorithm called **D3QN**. The algorithm combines Double Q-learning, a Dueling network, Prioritized Experience Replay, and a heuristic exploration policy. It outperforms A*, RRT, DQN, and DDQN in path quality, and achieves **~95% success** in dynamic environments after training.
