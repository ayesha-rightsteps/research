## Common Problem

All four papers use deep reinforcement learning to help UAVs navigate from a start point to a target while avoiding obstacles, but each solves only a part of the real problem — either one drone or many, either 2D or 3D, either static or dynamic obstacles, never all at once. No paper combines large-scale swarm coordination, a 3D environment, dynamic obstacles, and target assignment in a single framework. This gap — multi-UAV path planning in 3D with dynamic threats and simultaneous target assignment — is the open problem that future research needs to address.
